# Moi's Bridge Animal Feeds - Website

A professional, responsive website for Moi's Bridge Animal Feeds, a Kenyan animal feed supplier based in Moi's Bridge, Uasin Gishu County.

## Project Overview

This website showcases premium animal feed products and services for dairy, poultry, and pig farmers across Kenya. The site is designed to be user-friendly, mobile-responsive, and optimized for customer engagement and sales.

## Features

### Core Sections

1. **Navigation Bar** - Sticky navigation with smooth scrolling to all sections
2. **Hero Section** - Eye-catching hero image with call-to-action button
3. **Products Section** - Showcase of three main product categories:
   - Premium Dairy Meal (50kg bags)
   - Premium Layers Mash (25kg bags)
   - Premium Pig Starter Feed (20kg bags)

4. **About Section** - Company information with statistics and value propositions
5. **Services Section** - Six key services offered:
   - Bulk Delivery
   - Farmer Consultation
   - Flexible Payment Options
   - Fast Delivery
   - Easy Ordering
   - Quality Promise

6. **Contact Section** - Contact information and inquiry form
7. **Footer** - Quick links and social media

### Design Features

- **Responsive Design** - Mobile-first approach, works on all devices
- **Modern UI** - Clean, professional design with green and orange color scheme
- **Smooth Animations** - Scroll animations and hover effects
- **WhatsApp Integration** - Direct WhatsApp ordering buttons
- **Professional Typography** - Clear hierarchy and readability
- **Accessibility** - Semantic HTML and accessible color contrasts

## Setup Instructions

### Local Development

1. Clone or download the project files
2. Open `index.html` in your web browser
3. No build process or dependencies required - it's a static website

### Customization

#### Update Contact Information

Edit the contact section in `index.html`:
- Replace phone numbers with actual contact details
- Update email address
- Modify business hours as needed

#### WhatsApp Integration

Update the phone number in `script.js`:
```javascript
const phoneNumber = '254700000000'; // Replace with actual WhatsApp number
```

Format: Country code (254 for Kenya) + phone number without leading zero

#### Update Images

Replace the image paths in `index.html` with your own images:
- `hero-image.jpg` - Main hero banner (recommended: 1536x1024px)
- `dairy-feed.jpg` - Dairy product image (1024x1024px)
- `poultry-feed.jpg` - Poultry product image (1024x1024px)
- `pig-feed.jpg` - Pig product image (1024x1024px)

#### Modify Content

All text content can be easily edited in `index.html`. Key sections:
- Product descriptions and features
- About section information
- Services offered
- Contact details
